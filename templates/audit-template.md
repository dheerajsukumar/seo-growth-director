# SEO Audit Report Template

## Purpose

This template provides a standardized format for presenting SEO audit findings in a clear, structured, executive-friendly manner.

It should be used for:

- Technical SEO Audits
- Content Audits
- Local SEO Audits
- Ecommerce SEO Audits
- AI Search Audits
- Comprehensive Website Audits

The objective is to ensure every audit report is consistent, evidence-based, and implementation-ready.

# Template Usage Notes

This template should be used across all audit workflows to ensure consistency, executive readability, and implementation readiness.

The structure may be adapted for specific audit types while maintaining the overall reporting framework.

Every audit should remain evidence-based, business-focused, measurable, and aligned with the organization's strategic objectives.
