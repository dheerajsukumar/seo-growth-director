# SEO Growth Director
> Enterprise-grade Claude Skill for strategic SEO, AI Search, and organic growth engineering.

## Vision

Build the most trusted enterprise-grade Claude Skill for SEO strategy, AI Search optimization, and organic growth engineering—helping businesses make evidence-based decisions through structured analysis, repeatable workflows, and implementation-ready recommendations.

## Objectives

The SEO Growth Director is designed to:

- Deliver executive-level SEO strategy rather than generic SEO advice.
- Prioritize business outcomes over vanity metrics.
- Produce implementation-ready audits, roadmaps, and recommendations.
- Support Technical SEO, Semantic SEO, Local SEO, E-commerce SEO, and AI Search optimization.
- Follow repeatable workflows for consistent, high-quality outputs.
- Help teams make evidence-based SEO decisions.

## Why This Skill Exists

Modern SEO has evolved beyond keyword optimization. Businesses now need a strategic system that combines Technical SEO, Semantic SEO, Entity Optimization, AI Search, and business-focused decision making.

Most AI assistants provide generic SEO advice. SEO Growth Director is designed to operate like an experienced Head of SEO by:

- Understanding business objectives before recommending SEO actions.
- Following structured, repeatable workflows rather than ad hoc responses.
- Producing implementation-ready deliverables instead of high-level suggestions.
- Incorporating modern search practices including AI Search, Generative Engine Optimization (GEO), and topical authority.
- Prioritizing sustainable, evidence-based strategies over short-term ranking tactics.

## Project Architecture

SEO Growth Director follows a modular architecture to ensure maintainability, scalability, and consistent AI reasoning.

```text
README.md
        │
        ▼
CLAUDE.md
        │
        ▼
Identity Layer
        │
        ▼
Knowledge Layer
        │
        ▼
Workflow Layer
        │
        ▼
Template Layer
        │
        ▼
Generated Outputs
```

### Architecture Layers

| Layer | Purpose |
|--------|---------|
| README | Project documentation and overview |
| CLAUDE | Global orchestration and system instructions |
| Identity | Defines the role, mission, principles, and communication style |
| Knowledge | Contains domain expertise and SEO best practices |
| Workflows | Encodes repeatable processes and decision-making |
| Templates | Standardizes reports, briefs, audits, and deliverables |
| Outputs | Final responses generated for end users |
