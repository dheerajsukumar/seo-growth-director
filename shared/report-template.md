# Report Template

## Purpose

The Report Template defines the standard structure for all reports generated by the SEO Growth Director.

The objective is to ensure every report is consistent, professional, actionable, and aligned with business objectives regardless of the workflow that generated it.

---

# Core Philosophy

Every report should help stakeholders understand:

- What was evaluated
- What was discovered
- Why it matters
- What should be done
- What business outcomes are expected

Reports should enable informed decision-making rather than simply presenting data.

---

# Report Objectives

The Report Template enables the SEO Growth Director to:

1. Standardize report structure.
2. Improve executive communication.
3. Support technical implementation.
4. Improve readability.
5. Increase recommendation clarity.
6. Support stakeholder alignment.
7. Improve implementation success.
8. Maintain consistent documentation quality.

---

# Standard Report Structure

Every report should contain the following sections:

1. Executive Summary
2. Objectives
3. Scope
4. Methodology
5. Key Findings
6. Supporting Evidence
7. Priority Recommendations
8. Implementation Roadmap
9. Success Metrics
10. Next Steps

The report structure should remain consistent across all workflows.
---

# Executive Summary

Every report should begin with a concise Executive Summary.

The Executive Summary should include:

- Overall assessment
- Key findings
- Critical issues
- High-priority opportunities
- Estimated business impact
- Recommended next steps

The Executive Summary should enable decision-makers to understand the report within five minutes.

---

# Findings

Every finding should include:

- Finding title
- Description
- Supporting evidence
- Root cause
- Business impact
- SEO impact
- User impact
- AI Search impact (where applicable)

Every finding should be evidence-based and clearly explained.

---

# Recommendations

Each recommendation should include:

- Recommendation title
- Objective
- Implementation guidance
- Priority level
- Estimated effort
- Expected outcome
- Success metrics

Recommendations should be actionable and measurable.

---

# Implementation Roadmap

Recommendations should be grouped by implementation timeline.

## Immediate (0–30 Days)

Critical actions requiring immediate attention.

## Short Term (30–90 Days)

High-priority improvements with significant business value.

## Medium Term (3–6 Months)

Strategic improvements that strengthen long-term SEO performance.

## Long Term (6–12 Months)

Continuous optimization initiatives and future growth opportunities.

The roadmap should help stakeholders plan implementation realistically.
---

# Report Quality Checklist

Before presenting any report, validate that:

- The Executive Summary is concise and accurate.
- Objectives are clearly defined.
- Scope is documented.
- Findings are supported by evidence.
- Root causes have been identified.
- Recommendations are actionable.
- Priority levels are assigned consistently.
- Implementation roadmap is realistic.
- Success metrics are measurable.
- Next steps are clearly defined.

Every report should be reviewed for clarity, completeness, and business relevance before presentation.

---

# Report Validation

The SEO Growth Director should validate that every report:

- Aligns with business objectives.
- Follows the standard report structure.
- References the shared reasoning framework.
- Applies the shared scoring framework.
- Uses the shared priority matrix.
- Follows the recommendation framework.
- Meets the output standards.
- Avoids unsupported conclusions.
- Maintains consistent terminology from the SEO Glossary.

Reports should be suitable for both executive stakeholders and implementation teams.

---

# Guiding Principle

A report should lead to action.

The SEO Growth Director should produce reports that clearly explain the current situation, identify opportunities, prioritize improvements, and provide a practical roadmap for implementation.

Every report should help stakeholders make informed decisions and achieve measurable business outcomes.
