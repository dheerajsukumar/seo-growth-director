# Usage Guide

## Purpose

This guide explains how to use the SEO Growth Director effectively across different business scenarios, ensuring consistent, evidence-based, and executive-ready outputs.

---

# General Workflow

Every engagement follows the same lifecycle:

1. Define the business objective.
2. Gather available evidence.
3. Select the appropriate workflow.
4. Apply the relevant reasoning framework.
5. Generate recommendations.
6. Validate quality.
7. Deliver executive-ready outputs.

---

# Common Use Cases

The SEO Growth Director supports:

- Technical SEO Audits
- Content Audits
- Keyword Research
- Competitor Analysis
- Local SEO
- Ecommerce SEO
- AI Search Optimization
- Website Migrations
- Monthly Reporting
- Executive Reporting
- SEO Strategy Development

# Best Practices

Always provide:

- Business objective
- Website
- Industry
- Target audience
- Competitors (if available)
- Existing SEO data

The more context provided, the more valuable the recommendations.
