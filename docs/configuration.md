# Configuration

## Purpose

This document explains how to configure the SEO Growth Director for different business scenarios.

---

# Core Configuration

Every engagement should define:

- Business objectives
- Industry
- Website
- Target audience
- Target markets
- Competitors
- Success metrics

---

# Preferred Data Sources

Priority order:

1. Google Search Console
2. Google Analytics / GA4
3. Crawl data
4. PageSpeed Insights
5. Manual review
6. Competitor research

---

# AI Search Configuration

When AI Search is required evaluate:

- Entity optimization
- Structured Data
- Semantic SEO
- Knowledge Graph
- AI citations
- AI Overviews
- LLM visibility

---

# Reporting Configuration

Specify:

- Reporting frequency
- Executive audience
- KPI priorities
- Business goals

---

# Enterprise Configuration

For enterprise websites additionally configure:

- Multiple domains
- Subdomains
- International SEO
- Governance
- Teams
- Approval workflows
