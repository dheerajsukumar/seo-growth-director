# Best Practices

## Purpose

This guide documents recommended practices for using and maintaining the SEO Growth Director.

---

# Business-First Thinking

Always begin with:

- Business objective
- Customer needs
- Commercial priorities

Avoid starting with technical SEO in isolation.

---

# Evidence-Based Decisions

Recommendations should rely on:

- Analytics
- Search Console
- Crawl data
- Manual review
- Competitive research

Clearly distinguish facts from assumptions.

---

# Prioritization

Focus first on initiatives with the greatest combination of:

- Business impact
- SEO impact
- Implementation feasibility

Avoid treating every issue as equally important.

---

# Executive Communication

Present:

- Outcomes
- Risks
- Opportunities
- Priorities

Avoid excessive technical detail unless requested.

---

# AI Search

Treat AI Search as an extension of SEO.

Prioritize:

- Entities
- Semantic relationships
- Structured Data
- Authority
- Trust

Avoid chasing short-term tactics.

---

# Documentation

Keep documentation:

- Current
- Versioned
- Consistent
- Easy to navigate

---

# Repository Maintenance

When updating:

- Preserve folder structure.
- Update CHANGELOG.
- Run all tests.
- Review documentation.
- Verify integration.

---

# Guiding Principle

Long-term consistency is more valuable than rapid expansion.

Every enhancement should strengthen the overall system rather than increase complexity.
