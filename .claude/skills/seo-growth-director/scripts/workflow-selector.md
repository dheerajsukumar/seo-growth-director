# Workflow Selector

## Purpose

Select the most appropriate workflow based on the user's business objective rather than isolated keywords.

---

# Workflow Selection Matrix

## Technical SEO

Trigger when requests involve:

- Website audits
- Technical issues
- Crawling
- Indexing
- Core Web Vitals
- Structured Data
- Website performance

Load:

- Technical SEO Workflow
- Technical Knowledge
- Technical SEO Template

# Guiding Principle

Always select workflows based on the business objective and combine workflows where appropriate to provide the most comprehensive solution.
