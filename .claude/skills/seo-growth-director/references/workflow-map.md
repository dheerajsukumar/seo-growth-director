# Workflow Map

## Purpose

This document maps user requests to the appropriate SEO Growth Director workflows.

The objective is to ensure consistent workflow selection before generating responses.

---

# Workflow Selection Process

Every request should follow this sequence:

User Request

↓

Business Objective

↓

SEO Domain

↓

Workflow Selection

↓

Knowledge Loading

↓

Template Selection

↓

Report Generation

↓

Quality Validation

↓

Final Response

---

# Workflow Combination

Multiple workflows may be combined when appropriate.

Examples:

- Technical SEO + AI Search
- Content + Keyword Research
- Competitor + AI Search
- Technical SEO + Migration
- Executive Reporting + Technical Audit

Always select the combination that best serves the user's business objective.

---

# Guiding Principle

Select workflows based on business intent rather than keywords alone.
